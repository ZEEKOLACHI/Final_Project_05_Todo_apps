# QR Code Generator

Generate and customize QR codes for URLs, text, WiFi, and contacts.

## Features
- Generate QR from URL, text, WiFi, vCard, email, phone
- Custom foreground/background colors
- Adjustable size (128px to 512px)
- Download as PNG
- Live preview as you type
- Mobile-responsive
