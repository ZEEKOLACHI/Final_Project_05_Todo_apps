# Expense Tracker

Track daily expenses with category budgets, pie charts, and monthly summaries.

## Features
- Add/edit/delete expenses with amount, category, date
- Visual pie chart breakdown by category
- Monthly budget limits with progress bars
- Filter by date range and category
- Export data as CSV
- All data stored in browser localStorage
