# Markdown to PDF Converter

Live markdown editor with instant preview and one-click PDF export.

## Features
- Split-pane editor with live preview
- Supports full Markdown syntax (headings, bold, italic, code, lists, tables)
- Multiple themes (Light, Dark, Academic)
- One-click PDF download
- Responsive design
